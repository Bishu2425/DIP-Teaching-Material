# Teaching Material for DIP Class

### 1. Running the code

1. Clone this repository

   ```
   $ git clone https://github.com/oil-mcut/DIP-Teaching-Material.git
   ```

2. Setup environment

   ```
   $ cd DIP-Teaching-Material
   $ pip3 install -r requirments.txt
   ```

3. Running Jupyter notebook

   ```
   $ jupyter notebook
   ```
